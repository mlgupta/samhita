What is Permission Bundle?
  • To manage access more broadly than the permissions defined by AccessLevel, you can use PermissionBundle to group AccessLevels. PermissionBundles are stored persistently in the database, and can be used to represent higher level security settings which may be granted to users.
  • For example, you might group the Discover, Add Item, and Remove Item permissions together into a ModifyFolder Permission Bundle to more easily grant users the ability to modify folders.
 
How to Search Permission Bundle?
  • Enter search condition in 'Search Permission' either full Permission Bundle Name or the starting words can be entered to search for related Permission Bundle.
    (You can also use wild card characters '*' or '?')
  • The Permission Bundles will be displayed in the Permission Bundle List which match the entered text.
  • If the search condition doesn't match,a page will be displayed with text 'There is no item to display'.
 
What is Permission Bundle List?
  • Permission Bundle List consists of the Permission Bundles.
