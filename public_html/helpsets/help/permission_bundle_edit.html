How To Edit a Permission Bundle?
  • You can edit one Permission Bundle at a time.
  • Click on 'Edit Permission Bundle' Icon [ ] from the Permission Bundle List, Edit Permission Bundle Page appears with the previously saved data.

Description     Permission Bundle's description. Optional
Access Level     Access Levels are permissions.  
Available Permissions     Lists the available permissions.  
Selected Permissions     Lists the selected permissions. Required
Move '>'     Moves the selected permissions from 'Available Permission' to 'Selected Permissions'.  
Move All '>>'     Moves all the permissions from 'Available Permission' to 'Selected Permissions'.  
Remove '<<'     Moves all the permissions from 'Selected Permission' to 'Available Permissions'.  
Remove '<'     Moves the selected permissions from 'Selected Permission' to 'Available Permissions'.  
'Save' button     Press 'Save' to save the Permission Bundle, the Permission Bundle will be saved and Permission Bundle list will appear with the message in the status Bar as Permission Bundle 'ModifyFolder' successfully saved.  
'Cancel' button     Press 'Cancel' to abort editing Permission Bundle.
