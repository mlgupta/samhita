How to submit document to workflow?
  • In Folder/Document List select the document for workflow submission.
  • Click on the Submit to Workflow icon [ ] or select from menu File->Submit To Workflow.
  • Please note it is mandatory that the user's Workflow Acl must be set as well as valid.
  • If this is not the case,then the user must contact System Administrator.
  • User can view the status of his document by viewing Notification page.
