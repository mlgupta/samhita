How to upload a file or document?
  • In Folder/Document List click the folder where you would like to upload new file.
  • Click on the upload icon [ ] or select from menu File->Upload File.
 
  • Press 'Browse' to select the file to be uploaded from your system.
  • Select the file to be uploaded.
  • Enter the description. (optional)
  • Press 'Add' to add the file in file-list .
  • Press 'Remove' to remove selected file(s) from file-list.
  • Press 'Upload' to upload file(s) from file-list.
  • Press 'Cancel' to abort uploading.
