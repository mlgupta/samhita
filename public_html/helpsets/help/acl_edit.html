How To Edit an ACL?
  • You can edit one ACL at a time.
  • Click on 'Edit ACL' icon [ ] from the ACL List, Edit ACL page appears with default settings.

Add     Select option User/Group to be added as Access Control Entry. Required
'Add' button     To make the Access Control entry:
  • Click on the 'Add' button.
  • Permission for this entry will be listed.
  
'Find' button     To display User/Group list:
  • Click on the 'Find' button.
  • Depending on the 'Add' select option User or Group list will be displayed.
  • Select the User/Group.
  • Press 'Select' to select, the selected item will be displayed it the text before Add button.
  • Press 'Cancel' to abort.
 
'Remove' button     Select the Access Control Entry and Press 'Remove' to remove the Access Control from the ACL.  
'Save' button     Press 'Save' to save the modified Acl, the ACL will be saved and ACL list will appear with the message in the status Bar as "ACL 'Steffia' successfully saved."  
'Cancel' button     Press 'Cancel' to abort changes on the ACL.
