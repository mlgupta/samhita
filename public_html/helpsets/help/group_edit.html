How To Edit a Group?
  • You can edit one group at a time.
  • Click on 'Edit Group' icon [ ] from the Group List, Edit Group Page appears with previously saved data.

Name Min Length Max Length Description Required /Optional
Group Name 5 20 Enter unique Group Name. Non-Editable
Description - - Group's description. Optional
Access Control List - - Access control list that will be applied by default to any objects created by this Group.
  • Click on look-up next to Access Control List.
  • Access Control List will be displayed.
  • Select the Access Control List.
  • Press 'Select' to select.
  • Press 'Cancel' to abort.
Required
Add Users/Groups     Select option either User/Group to be included in this Group. Optional
'Add From User List' button     To add user to the Group.
  • Click on the button 'Add From User List'.
  • User list will be displayed.
  • Select the User. (multiple selection is allowed.)
  • Press 'Select' to add.
  • Press 'Cancel' to abort.
Optional
Add From Group List     To add groups to the Group.
  • Click on the button 'Add From Group List'.
  • Group list will be displayed.
  • Select the Group. (multiple selection is allowed.)
  • Press 'Select' to add.
  • Press 'Cancel' to abort.
Optional
Remove From List     Select the Users/Groups from the Group List.
  • Select the Group. (multiple selection is allowed.)
  • Click on the button 'Remove From List'.
Optional
'Save' button     Press 'Save' to save changes done on the Group, the Group will be saved and Group list will appear with the message in the status Bar as "Group 'Marketing' successfully saved".  
'Cancel' button     Press 'Cancel' to abort changes done on Group.
