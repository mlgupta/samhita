How to Encrypt document?
  • In Folder/Document List select the document you would like to encrypt.
  • Click on the 'Encrypt Document' icon [ ].
  • Enter the Encryption password to encrypt the document
  • Press 'OK' if you would like to encrypt the document.
  • Press 'Cancel' to abort encryption.
 
How to Decrypt document?
  • In Folder/Document List select the document you would like to decrypt.
  • Click on the 'Decrypt Document' icon.
  • Enter the encryption password to decrypt the document.
  • Press 'OK' if you would like to decrypt the document.
  • Press 'Cancel' to abort decryption.
