How to Search Users?
  • Enter search condition in 'Search by User Name' either full User Name or the starting words can be entered to search for related Users.
    (You can also use wild card characters '*' or '?')
  • The Users will be displayed in the User List which matches the entered text.
  • If the search condition doesn't matches page will be displayed with text 'There is no item to display'.
 
What is User List?
  • User List consists of the list of account holders in the application.
