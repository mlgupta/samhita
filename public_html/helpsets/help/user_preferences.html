What is User Preference?
  • User Preference is the profile of the logged-in user where information related to his profile is displayed.
  • The preferences of the logged-in user can be set like
    • 'Login Password'.
    • 'Watch Documents'.
    • 'Encrypt/Decrypt Password'.
    • 'Permitted tree access level'.
    • 'Items to be displayed per page'.
    • 'Themes'.
    • 'Open Document In New Window '.

Encryption/decryption Set Password - - Facilitates the logged-in user to set or change the encryption/decryption password.
  • Click on the 'Set/Change Encrypt/Decrypt Password' button.
  • Encrypt/decrypt password page will be displayed.
  • Enter the 'Password' and 'Confirm Password'.
  • Press 'OK' to set/change the password.
  • Press 'Cancel' to abort.
-
Set URL Encrypt/Decrypt Password - - Facilitates the System-Admin to set the URL encryption/decryption password.
  • Click on the 'Set URL Encrypt/Decrypt Password' button.
  • URL Encrypt/decrypt password page will be displayed.
  • Enter the 'Password' and 'Confirm Password'.
  • Press 'OK' to set the password.
  • Press 'Cancel' to abort.
For System-Admin access only, once set, cannot be altered.
