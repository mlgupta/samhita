How To Create an ACL?
  • You can create one ACL at a time.
  • Click on 'New ACL' icon [ ] from the ACL List, New ACL page appears with the default settings.
Create New ACL

General ACL Information:
Name Min Length Max Length Description Comment
ACL Name 5 20 Enter unique ACL Name. Required
Description     Enter ACL description. Optional
Acl Type - - Select the type of ACL to create.
  • "NonAdapter ACL" represents a general ACL (always present).
  • "Approver WorkFlow" represents an ACL for approval purpose (always present).
  • "PeopleSoft Financials" represents an ACL for PS Fin adapter .
  • "PeopleSoft HR" represents an ACL for PS HR adapter .
  • "SAP" represents an ACL for SAP adapter .
  • Apart from "NonAdapter ACL" and "Approver WorkFlow" only those adapters that are enabled will be present in this list.
Required (by default "Non Adapter ACL is selected otherwise")
Owner Name 5 20 Enter Owner Name who is owning this ACL. Required
Securing ACL Name     ACL name which will secure this ACL.
  • Click on lookup next to Securing ACL Name.
  • Access Control List will be displayed.
  • Select the Access Control List.
  • Press 'Select' to select.
  • Press 'Cancel' to abort.
Required
Add     Select option User/Group to be added as Access Control Entry. Required
'Add' button     To make the Access Control entry:
  • Click on the 'Add' button.
  • Permission for this entry will be listed.
 
'Find' button     To display User/Group list:
  • Click on the 'Find' button.
  • Depending on the 'Add' select option User or Group list will be displayed.
  • Select the User/Group.
  • Press 'Select' to select, the selected item will be displayed it the text before Add button.
  • Press 'Cancel' to abort.
 
'Save' button     Press 'Save' to save the newly created Acl, the ACL will be saved and ACL list will appear with the message in the status Bar as "ACL 'SteffiaACL' successfully saved."  
'Cancel' button     Press 'Cancel' to abort new ACL creation.
