How to apply ACL?
  • In Folder/Document List select the folder/document on which you would like to apply ACL.
  • You can select multiple folder/document.
  • Click on the 'Apply ACL' icon [ ] or select from menu File->Apply ACL.
  • Select an ACL to be applied on Folder/Document or search an ACL by giving the keyword.
    (Enter a word in the text box and click Search. You can also use wild card "*")
  • Apply Recursively - Check this option if you would like to apply ACL on sub Folders and documents.
  • Press 'Select' to apply ACL on the Folder/Document.
  • Press 'Cancel' to abort applying ACL.
