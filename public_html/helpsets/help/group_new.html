How To Create a Group?
  • You can create one group at a time.
  • Click on New Group icon [ ] from the Group List, New Group Page appears with default settings.

Add From User List     To add user to the Group.

Optional
'Add From Group List' button     To add groups to the Group.
  • Click on the button 'Add From Group List'.
  • Group list will be displayed.
  • Select the Group. (multiple selection is allowed.)
  • Press 'Select' to add.
  • Press 'Cancel' to abort.
Optional
'Remove From List' button     Select the Users/Groups from the Group List.
  • Select the Group. (multiple selection is allowed.)
  • Click on the button 'Remove From List'.
Optional
'Save' button     Press 'Save' to save the New Group, the Group will be saved and Group list will appear with the message in the status Bar as "Group 'Marketing' successfully saved".  
'Cancel' button     Press 'Cancel' to abort New Group creation.
