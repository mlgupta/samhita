What is Voucher Status List?
  • Voucher status list displays the list of vouchers that are submitted to PeopleSoft workflow from Samhita with their respective current status .
  • Every voucher in Samhita can have one of the following 3 status :
    • InQueue : When a user submits vouchers or folders containing vouchers to PeopleSoft workflow, then the initial status of all the submitted vouchers is set to "InQueue".
    • InProcess : When a user clicks on "GetNextImage" button to get the image of next voucher from samhita workflow queue, the status of the voucher gets changed to "InProcess".
    • Processed : When a user successfully saves the voucher in PeopleSoft, the voucher status changes from "InProcess" to "Processed".
  • Inorder to reset voucher status from "InProcess" to "InQueue", select the desired vouchers and click on the "Reset" [ ] icon.

How to search Voucher List?
  • Voucher search can be done with following criteria:
    • Status : Select the current status of the vouchers to search
    • Zone : Select the zone of vouchers that is ACL name corresponding to People Soft.
    • From Date :
      • Enter the 'From' time/date for vouchers as search criteria.
    • To Date :
      • Enter the 'To' time/date for vouchers as search criteria.
    • Upon entering the search criteria , press "Go".
    • The voucher(s) matching the search criteria will be displayed in the voucher list.
    • If the search condition doesn't match, a page will be displayed with text 'There is no item to display'.
