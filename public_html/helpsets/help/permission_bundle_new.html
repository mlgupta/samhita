How To Create a Permission Bundle?
  • You can create one Permission Bundle at a time.
  • Click on 'New Permission Bundle' icon [ ] from the Permission Bundle List, New Permission Bundle Page appears with default settings.
Create New Permission Bundle
 
General Permission Bundle Information:
Name Min Length Max Length Description Comment
Permission Bundle Name 5 20 Enter unique Permission Bundle Name. Required
Description     Permission Bundle's description. Optional
Access Level     Access Levels are permissions.  
Available Permissions     Lists the available permissions.  
Selected Permissions     Lists the selected permissions. Required
Move '>'     Moves the selected permissions from 'Available Permission' to 'Selected Permissions'.  
Move All '>>'     Moves the entire permissions from 'Available Permission' to 'Selected Permissions'.  
Remove '<<'     Moves the entire permissions from 'Selected Permission' to 'Available Permissions'.  
Remove '<'     Moves the selected permissions from 'Selected Permission' to 'Available Permissions'.  
'Save' button     Press 'Save' to save the newly created Permission Bundle, the Permission Bundle will be saved and Permission Bundle list will appear with the message in the status Bar as "Permission Bundle 'BundleName' successfully created."  
'Cancel' button     Press 'Cancel' to abort new Permission Bundle creation.
