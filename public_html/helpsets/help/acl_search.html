How to Search ACL's?
  • Enter search condition in 'Search ACL' either full ACL Name or the starting words can be entered to search for related ACL's.
    (You can also use wild card characters '*' or '?')
  • The ACL will be displayed in the ACL List which match the entered text.
  • If the search condition doesn't match,a page will be displayed with text 'There is no item to display'.
 
What is Acl List?
  • ACL List consists of the ACL's.
