How To Edit a User?
  • You can Edit one user at a time.
  • Click on 'Edit User' Icon [ ] from the User List, Edit User Page appears with previously saved data.

 Optional (if not specified default will be taken.)

Default Document Acl  The ACL that will be applied to any documents the user creates.

  • Press 'Cancel' to abort.
Optional (if not specified default will be taken.)
Default Folder Acl  The ACL that will be applied to any folders the user creates.
  • Click on look-up next to Access Control List.
  • Access Control List will be displayed.
  • Select the Access Control List.
  • Press 'Select' to select.
  • Press 'Cancel' to abort.
Optional (if not specified default will be taken.)
Workflow Acl  The ACL that contains users who will have to approve/reject document submitted to workflow by the user.
  • Click on look-up next to Access Control List.
  • Access Control List related to workflow will be displayed
  • Select the Access Control List.
  • Press 'Select' to select.
  • Press 'Cancel' to abort.
Required,if the user wishes to enable workflow. (if not specified none will be taken.)
Language  Select the language from the list that this user's client applications will use while interacting.Required
Character Set  Select the Character set required to display the selected language. Required
Locale  Select the Locale base on the selected language.Required
Time Zone  Select the User's time zone.Required
'Save' button  Press 'Save' to save the changes done on the User, the User will be saved and User list will appear with the message in the status Bar as "User 'Steffia' successfully saved".  
'Cancel' button  Press 'Cancel' to abort changes done on User.  

Note
  • A user's home folder is the default directory where a user starts when logging into the application.
  • The Email User Profile specifies the user's mail folder location and address.
  • Unless you change the default definitions, a user's e-mail folder is created under the home folder.
    For example, /home/jsmith/mail.
