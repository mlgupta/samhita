How To Use Permission?
Permissions are the List of permission Bundles.
 
How to Assign Permissions:
Permissions are the List of permission Bundles which are assigned to the Access Control Entry for the selected ACL. You can select the permission bundles from Available Permission Bundle list to the Selected Permission Bundle list and vice-versa to grant or revoke the permissions.
