How to see Property of Folder/Document?
  • In Folder/Document List select the folder/document for which you would like to see the Property.
  • You can select multiple folders/documents.
  • Click on the Property icon [ ] or select from menu File->Property.
  • You can rename the folder/document or apply another ACL to it.
  • Press 'OK' to save the changes made.
  • Press 'Cancel' to abort 'Rename' or applying 'ACL'.
