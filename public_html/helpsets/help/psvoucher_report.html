What is Voucher Report?
  • Voucher report is a bar chart representing the current status of vouchers.
  • The voucher report displays :
    • Total : the total number of vouchers available.
    • InQueue : the number of vouchers available in "InQueue" state.
    • InProcess : the number of vouchers available in "InProcess" state.
    • Processed : the number of vouchers available in "Processed" state.

How to search in voucher report?
  • Voucher report search can be done with following criteria:
    • Zone : Select the zone of vouchers that is ACL name corresponding to People Soft.
    • From Date :
      • Enter the 'From' time/date for vouchers as search criteria.
    • To Date :
      • Enter the 'To' time/date for vouchers as search criteria.
    • Upon entering the search criteria , press “Generate” button.
    • The appropriate voucher report matching the search criteria will be displayed in the form of bar chart.
