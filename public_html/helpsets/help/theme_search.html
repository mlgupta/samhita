How to Search Themes?
  • Enter search condition in 'Search by Theme' either full Theme Name or the starting words can be entered to search for related Themes.
    (You can also use wild card characters '*' or '?')
  • The Themes will be displayed in the Theme List which match the entered text.
  • If the search condition doesn't match,a page will be displayed with text 'There is no item to display'.
 
What is Theme list?
  • Theme List is list of supplied and customizable Themes.
